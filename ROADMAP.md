# StructDiff Studio Roadmap

## 1. Product Foundation

- Neutral application branding
- XML/HTML folder and ZIP comparison
- Native Windows diff bundle
- HTML dashboard and side-by-side report
- Demo sample data

## 2. Structured Diff Engine

- XPath-style change output for XML and HTML
- Attribute-level diff summary
- Missing and extra file detection
- Rule-based normalization presets
- Ignored change count in reports

## 3. Multi-Format Support

- JSON parser
- YAML parser
- CSV parser
- Plain text mode
- Plugin-like parser interface

## 4. Developer Experience

- CLI mode
- JSON summary export
- CSV export
- Benchmark command
- GitHub Actions build pipeline

## 5. Portfolio Polish

- Screenshot gallery
- Short demo video
- Benchmark table
- Architecture diagram
- Public release package
